# Quiz-App


A simple and interactive Quiz App built with React.js. It allows users to answer multiple-choice questions, provides instant feedback (correct/wrong answers), tracks progress, and shows the final score with a reset option.
<br>
<br>
Home / First Question Screen
<br>
Displays the quiz title and the first question with four multiple-choice options.<br>
A “Next” button is shown to move to the next question.<br>
Progress counter is displayed (e.g., “1 of 10 questions”).<br>

Answer Selected (Correct Answer Highlight)
<br>
When the user selects the correct answer, the option is highlighted in green.<br>
Other options remain unchanged.<br>
The quiz locks the question until the user presses “Next”.<br>

<br>
Answer Selected (Wrong Answer Highlight)
<br>
If the user selects a wrong answer, the selected option is highlighted in red.<br>
The correct answer is automatically highlighted in green.<br>
User can proceed only after pressing “Next”.<br>

Final Result Screen
<br>
At the end of the quiz, the user’s final score is displayed (e.g., You Scored 5 out of 10).<br>
A Reset button is provided to restart the quiz from the beginning.<br>
